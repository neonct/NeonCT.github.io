# NeonCT 官网网站

## NeonCT 是什么

NeonCT（霓虹芒果）是一门免费开源的轻量级游戏开发框架，由豆焰主导开发工作。 期望在所有主流电脑平台（OSX、Windows和Linux）都能支持运行。
